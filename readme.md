## 리액트 교과서 정리


### 개요

![리액트 교과서](./assets/cover.jpg)

- 순차적으로 책에서 설명하는 내용과 샘플 코드를 정리
- 실행 가능한 코드를 챕터마다 구성하며, 일부 코드는 작성한 한 상태
- 첨부된 코드의 구동을 위해서는 로컬 서버가 필요할 수 있음
